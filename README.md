# \# 🏥 Smart Hospital IoT System





A Secure Healthcare IoT System built using \*\*React.js, Node.js, Express.js, MySQL, Arduino UNO R4 WiFi, ESP32-CAM, Fingerprint Authentication, Face Recognition, and IoT Sensors.\*\*



\---



\# 📖 Project Overview



The Smart Hospital IoT System is a secure IoT-based healthcare solution that enables patient authentication, doctor access, and real-time monitoring of vital signs. The system integrates biometric authentication, face recognition, IoT sensors, and a React-based web application to improve healthcare management, patient safety, and secure access to medical records.



\---



\# ✨ Features



\- 👤 Patient Registration

\- 🔐 Patient Login using Patient ID \& Password

\- 👉 Fingerprint Authentication

\- 📷 Face Recognition using ESP32-CAM

\- ❤️ Live Pulse Monitoring

\- 📡 Real-time Sensor Monitoring
